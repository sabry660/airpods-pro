# AirPods Pro Website
This is a clone of the official [Apple AirPods Pro](https://www.apple.com/in/airpods-pro/) website.
### Built with
 HTML
 CSS
 Vanilla JS
 ScrollMagic Library
 GSAP Animation Library
##  Main Focus
My main focus was on recreating the main animation with the highest accuracy.
